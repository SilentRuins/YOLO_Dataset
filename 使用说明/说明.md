# 将视频制作成标准数据集并训练模型
1. 准备若干个可以用于制作数据集的视频(可以自己准备要识别的物体，以各种角度拍摄该物体。我使用了两段拍摄手机充电头和固体胶的视频)

2. 将视频放到`make_dataset\数据集视频`文件夹下

3. 运行`make_data.py`截取视频帧，该脚本会把图片放到`img`文件夹下，此时`img`就是未标注和格式化的数据集

4. 运行`pick.py`，该脚本从`img`文件夹中按固定间隔抽取少量图片，复制到`predata`文件夹下。该文件夹为yolo标准数据集文件格式，用于训练自动标注需要的模型

5. 在`predata\labels`文件夹下创建`classes.txt`文件，并写入类别，使用**labelimg**对`predata`数据集手动标注

6. 标注完成后就可以使用该数据集在yolo训练一个模型啦。把`predata`复制到yolo的`datasets`文件夹下，同时在yolo源文件`ultralytics\cfg\datasets`文件夹下配置.yaml文件，写好代码就可以开始训练了
```py
model = YOLO(r"yolo26n.pt")  
train_results = model.train(  
    data="predata.yaml",  # 数据集配置文件路径  
    epochs=30,  # 训练周期数  
    imgsz=960,  # 训练图像尺寸  
    batch=-1,   #每批数据数量，-1为自动匹配  
)
```

7. 在yolo源文件`runs\detect`文件夹中找到训练好的.pt模型文件，使用该模型对所有数据进行预测，记得调整predict()参数`save_txt=True`保存预测出的数据标签
```py
model = YOLO(r'.\runs\detect\train\weights\best.pt')  
result = model.predict(  
    source='img',  # 图片源  
    conf=0.2,  # 置信度阈值，因为这个模型不精确，设置小一点
    save=True,  # 保存结果  
    save_txt=True, # 保存标注结果  
    show=False, # 预测时展示结果  
)
```

8. 把预测出的数据标签复制到`make_dataset\labels`下，记得把`predata`手动标注的`labels`也复制过来

9. 这时候`img+labels`已经是完整的数据集了，但是自动标注并不完全准确，所以还需要使用**labelimg**对所有数据手动标注

10. 当所有标注手动标注完成后，就可以使用`format.py`将数据集标准化了

11. 得到了自己的标准数据集，为什么不训练个模型试试呢？
```py

    imgsz=960,  # 训练图像尺寸  
    batch=-1,   #每批数据数量，-1为自动匹配  
)
```

```py
# 使用训练好的模型预测
model = YOLO(r'.\runs\detect\train-6\weights\best.pt')
